Brimstone, o controlador veterano

Brimstone, o controlador veterano, é um dos agentes mais versáteis do VALORANT. Suas habilidades, baseadas em fogo e fumaça, permitem que ele controle o campo de batalha de forma estratégica e apoie sua equipe de diversas maneiras.

Habilidades

  • Sinalizador de Estímulo: Ativa instantaneamente um sinalizador que cura aliados e aumenta a taxa de disparo por um curto período. Ideal para iniciar um round ou recuperar a saúde de aliados feridos.
  • Incendiário: Lança uma granada incendiária que cria uma zona de fogo que causa dano aos inimigos. Perfeito para negar espaço, limpar cantos e atrasar o avanço inimigo.
  • Sky Smoke: Cria uma nuvem de fumaça em um local escolhido no mapa. A fumaça pode ser usada para bloquear a visão dos inimigos, criar rotas de fuga ou isolar áreas.
  • Ataque Orbital: A habilidade definitiva de Brimstone. Um raio de laser desce do céu, causando dano em uma área de efeito. Pode ser usado para limpar sites de bomba, eliminar inimigos agrupados ou proteger a equipe.

Como jogar com Brimstone

  • Controlador: A principal função de Brimstone é controlar o campo de batalha usando suas habilidades para negar espaço e criar oportunidades para sua equipe.
  • Visão de jogo: É fundamental ter uma boa visão de jogo para usar as habilidades de Brimstone de forma eficaz. Antecipe os movimentos dos inimigos e use a fumaça para bloquear suas linhas de visão.
  • Comunicação: A comunicação é essencial para qualquer agente, mas é ainda mais importante para Brimstone. Coordene suas habilidades com seus aliados para garantir o sucesso da equipe.
  • Flexibilidade: Brimstone é um agente muito versátil e pode se adaptar a diferentes situações. Experimente diferentes posições e combinações de habilidades para encontrar o que funciona melhor para você.

Dicas e Truques

  • Use a fumaça de forma estratégica: A fumaça pode ser usada para criar muitas situações diferentes. Use-a para bloquear a visão dos inimigos, criar rotas de fuga, isolar áreas, ou até mesmo para confundir os inimigos.
  • Combine as habilidades: Combine as habilidades de Brimstone para criar sinergia e maximizar o seu impacto. Por exemplo, use a fumaça para bloquear a visão dos inimigos e, em seguida, lance o incendiário para limpar o local.
  • Ajuste a altura da fumaça: A altura da fumaça pode ser ajustada para diferentes situações. Uma fumaça baixa pode ser usada para bloquear a visão dos inimigos sem revelar sua posição, enquanto uma fumaça alta pode ser usada para criar uma cobertura completa.
  • Use o sinalizador de estímulo com sabedoria: O sinalizador de estímulo é uma habilidade poderosa, mas deve ser usada com cuidado. Use-o para recuperar a saúde de aliados feridos ou para aumentar a taxa de disparo de um jogador que está prestes a iniciar um duelo.

Brimstone é um agente essencial para qualquer equipe. Com suas habilidades versáteis e capacidade de controlar o campo de batalha, ele pode mudar o rumo de uma partida.
